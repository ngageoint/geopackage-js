new GeoPackageConnection( filePath ) → {Promise.<module:db/geoPackageConnection~GeoPackageConnection>}

Description

Creates a connection to the SQLite file and when connected, returns a promise that resolves the connection. This will create a {module:db/sqliteAdapter~Adapter} if running in node and the FORCE_SQLJS environment variable is not set. This will create a {module:db/sqljsAdapter~Adapter} if running in the browser or the FORCE_SQLJS environment variable is set
