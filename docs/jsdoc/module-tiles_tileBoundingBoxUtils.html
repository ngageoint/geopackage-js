<static> webMercatorTileBox( webMercatorBoundingBox, zoom ) → {Object}

Description

Calculate the bounds in tile coordinates that covers the given bounding box at the given zoom level. The result object contains the keys minX, maxX, minY, and maxY, which are tile column and row values in the XYZ tile scheme.

Parameters
Name Type Description

bounds in EPSG:3857 coordinates (meters)
